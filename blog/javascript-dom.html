Control flow and loops Control flow is the order that code is executed and runs in order from the first line in the file to the last line, unless conditionals or loops are introduced to change the flow Loops are used to repeat a particular set of code until a certain condition is met. Consider doing the dishes, there are a clear series of tasks that need to be completed in a certain order, run the water, get the dish soap etc. Think of the scrubbing phase like a loop, you can explain how to scrub (which is like an function in code, part of the control flow) but you need to make sure what you’re scrubbing is clean before you move on, this is where the loop comes in, where you need to keep repeating the ‘scrub’ function until the condition is met ‘clean plate’ DOM – Document Object Model The DOM is a programming interface that represents HTML as a tree structure so that programs (JavaScript) can interact with it.
